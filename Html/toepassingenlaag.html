Toepassingenlaag: Apps en API's

Apps

Het woord "App" is een afkorting van "applicatie", wat betekent: toepassing. Het betreft een programma dat op een apparaat, zoals een computer of telefoon, is geïnstalleerd. Dit kunnen spellen zijn, sociale media, maar ook bijvoorbeeld google drive. Uiteindelijk zal jouw computer of telefoon deze programma's weergeven

API's

API staat voor Application programming interface. Dit maakt het mogelijk dat twee apps met elkaar kunnen communiceren. Op deze manier kan jouw weer app exact zien hoeveel graden het morgen wordt, dankzij een verbinding met de server van een weerservice.
